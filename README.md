# XMedia Converter

A local media converter built with **Next.js** + **pnpm**.
- **Video/Audio** conversions use **FFmpeg**
- **Images** conversions use **Sharp** (FFmpeg not required for images)

---

## Requirements

- **Node.js 18+** (recommended **20**)
- **pnpm**
- **FFmpeg** (only for audio/video)

---

## 1) Install Node.js (18+ / recommended 20)
